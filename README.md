मूढमते ।
सम्प्राप्ते सन्निहिते काले
नहि नहि रक्षति डुकृङ्करणे ॥

يا أيها الذين آمَنوا آمِنوا بالنورين أنزلناهما 
 يتلوان عليكم آياتي ويحذرانكم عذاب يوم عظيم

ਹਰਿ ਇਕੋ ਮੇਰਾ ਸਜਣੋ ਹਰਿ ਇਕਸੈ ਨਾਲਿ ਮੈ ਸੰਗੁ
